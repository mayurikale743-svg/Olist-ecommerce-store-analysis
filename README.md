# 🛒 Olist E-Commerce Store Analysis

## 📌 Project Overview

This project analyzes the Brazilian Olist E-Commerce dataset to uncover business insights related to sales performance, customer satisfaction, delivery efficiency, and payment behavior. The analysis was performed using SQL, Excel, Tableau, and Power BI to build interactive dashboards that support data-driven decision-making.

---

## 🎯 Business Objective

The objective of this project is to help stakeholders:

- Monitor overall business performance
- Identify top-performing product categories
- Analyze customer purchasing behavior
- Track delivery performance
- Measure customer satisfaction
- Support strategic business decisions through interactive dashboards

---
